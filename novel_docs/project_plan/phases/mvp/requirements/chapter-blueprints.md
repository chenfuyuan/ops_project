# 章节蓝图

## Goal

把阶段性大纲拆分成一组可逐章推进的蓝图单元，为正式章节写作提供清晰但不过度细化的计划层输入。

## User Value

作者可以在写章之前先看到章节目标和推进节奏，减少长篇创作时“下一章写什么”的空档。

## Success Criteria

- 蓝图需求明确表达每章或每批章节需要承担的叙事作用。
- 蓝图结果能够作为章节生成的直接上游输入。
- 卡片仍停留在需求边界，不提前展开蓝图数据结构或算法细节。

## Scope

- 从大纲到章节蓝图的转换需求
- 蓝图需要承载的核心章节目标与节奏信息
- 批次规划与续写衔接的用户价值

## Non-goals

- 蓝图字段 schema
- 自动优化算法
- 审计与修订规则
