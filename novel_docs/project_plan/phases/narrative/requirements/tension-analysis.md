# 张力分析

## Goal

持续观察章节节奏与张力走势，帮助团队判断作品是否在关键阶段保持足够的阅读驱动力。

## User Value

作者能够更早发现剧情推进过平、重复或失衡的章节，而不是等整段内容写完后才凭感觉返工。

## Success Criteria

- 系统具备衡量章节张力表现的独立需求边界。
- 张力结果能够被作者查看，并帮助判断后续是否需要调整。
- 卡片不提前落入具体评分算法或可视化实现方案。

## Scope

- 对章节节奏和张力表现的衡量需求
- 张力异常的识别和反馈需求
- 与后续工作台展示的衔接边界

## Non-goals

- 评分公式设计
- 风格检测
- 自动修订逻辑

## Dependencies / Prerequisites

- Phase 2 质量治理基线
