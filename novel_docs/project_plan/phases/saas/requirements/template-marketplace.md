# 模板市场

## Goal

支持作品模板或创作模板的发现、分发和复用，让平台能够积累可共享的创作资产。

## User Value

用户可以更快开始新项目，也能从已有模板中获得灵感和复用价值。

## Success Criteria

- 模板分发和发现被表达为独立需求边界。
- 该需求与账户、配额能力之间的依赖关系清晰。
- 卡片不提前定义市场排序、评分机制或内容审核实现。

## Scope

- 模板发布、浏览和复用的产品需求
- 模板作为平台资产的基本管理边界
- 与账户和用量管理的协同关系

## Non-goals

- 推荐算法
- 社区运营策略
- 审核工作流细节

- [usage-quotas](usage-quotas.md)
