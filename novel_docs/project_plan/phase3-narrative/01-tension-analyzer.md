# 张力分析器 (TensionAnalyzer)

## 概述

为每章生成 0-10 的张力评分，监控叙事节奏，预警异常模式。张力评分是衡量"这一章好不好看"的量化指标。

## 评分维度

总分 10 分，5 个维度各 2 分：

| 维度 | 满分 | 评估内容 |
|------|------|---------|
| 冲突强度 | 2 | 角色间冲突、角色与环境冲突、内心冲突的激烈程度 |
| 悬念密度 | 2 | 未解决问题数量、新悬念引入、章末钩子强度 |
| 信息揭示 | 2 | 新信息的重要程度、对读者认知的影响、秘密揭露 |
| 情感波动 | 2 | 角色情感变化幅度、读者共情强度、情感转折 |
| 节奏变化 | 2 | 与前章的对比、场景切换频率、叙事速度变化 |

## 实现方式

张力评分由 Reflector 在 Commit 阶段提取，扩展 StateDelta 输出。

评分结果包含以下字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| total | float | 0-10 总分 |
| conflict_intensity | float | 0-2 冲突强度 |
| suspense_density | float | 0-2 悬念密度 |
| information_reveal | float | 0-2 信息揭示 |
| emotional_fluctuation | float | 0-2 情感波动 |
| pacing_variation | float | 0-2 节奏变化 |
| analysis | str | 评分理由（简短文本） |

Reflector 提示词在现有状态提取提示词末尾追加张力评分要求，要求 LLM 从 5 个维度各打 0-2 分，并对低分维度给出简短理由。输出为结构化 JSON。

## 预警系统

每章 Commit 后检查张力模式，TensionWarningChecker 负责检测以下异常：

1. 连续低张力：最近 3 章张力均低于 3，提示节奏可能拖沓，建议引入冲突或悬念
2. 连续高张力：最近 3 章张力均高于 8，提示读者可能疲劳，建议安排缓冲章节
3. 节奏单调：最近 10 章张力标准差低于 1.0，提示节奏缺乏起伏
4. 偏离蓝图：实际张力与蓝图规划的 suspense_density 偏差超过 4 分

## 预警注入

张力预警注入 Planner 的输入，格式为最近张力趋势数组 + 预警信息 + 建议，让 Planner 在规划下一章时参考。

## 前端

### 张力曲线图

- 使用 ECharts 折线图
- X 轴：章节号
- Y 轴：张力分（0-10）
- 实际张力曲线（蓝色实线）
- 蓝图规划的 suspense_density 参考线（灰色虚线，1-5 映射到 2-10）
- 预警区域高亮（连续低张力区间标红色背景，连续高张力标橙色背景）
- 鼠标悬停显示详细评分（5 个维度）
- 点击数据点跳转到对应章节

### 张力面板

写作工作台右栏新增"张力"Tab：
- 当前章节张力评分（雷达图，5 个维度）
- 与前章对比
- 预警信息（如有）
