# 前端界面设计

## 页面结构

```
/                           → 首页（小说项目列表）
/novels/new                 → 创建新项目
/novels/:id                 → 项目仪表盘
/novels/:id/outline         → 大纲编辑
/novels/:id/blueprints      → 章节蓝图
/novels/:id/write/:chapter  → 写作工作台
/novels/:id/characters      → 角色管理
/novels/:id/foreshadowing   → 伏笔台账
/settings/llm               → LLM 模型配置
```

## 页面详细设计

### 首页 — 小说项目列表

**布局：** 卡片网格

**每张卡片显示：**
- 小说标题
- 类型标签（玄幻/仙侠/都市等）
- 语言标签（中文/英文）
- 进度：已完成章节数 / 总章节数
- 进度条
- 最后更新时间
- 操作：打开 / 删除

**顶部操作栏：**
- "创建新项目"按钮
- 搜索框
- 排序（最近更新 / 创建时间 / 名称）

### 创建新项目

**表单字段：**
- 标题（必填）
- 前提/简介（必填，多行文本框，建议 100-500 字）
- 类型（下拉选择：玄幻/仙侠/都市/科幻/悬疑/言情/武侠/LitRPG/horror/other）
- 语言（中文/英文）
- 总章节数（数字输入，默认 200）
- 每章目标字数（数字输入，默认 3000）

### 项目仪表盘

**布局：** 左侧导航 + 右侧内容区

**左侧导航：**
- 仪表盘（概览）
- 大纲
- 蓝图
- 章节列表
- 角色
- 伏笔
- 设置

**仪表盘内容：**
- 项目基本信息卡片
- 写作进度（已完成/总章节，进度条）
- 最近生成的章节列表（最近 5 章）
- 快捷操作：生成下一章、查看大纲、编辑蓝图
- 活跃伏笔数量 / 已解决伏笔数量

### 大纲编辑页

**布局：** 步骤导航 + 内容区

**步骤导航（横向）：**
Step 1: 核心种子 → Step 2: 角色动力 → Step 3: 角色状态 → Step 4: 世界观 → Step 5: 三幕结构

每步显示状态：未生成 / 已生成 / 已编辑

**内容区（按步骤切换）：**

Step 1 核心种子：
- 文本显示区（只读模式 / 编辑模式切换）
- 操作按钮：生成 / 重新生成 / 编辑 / 保存

Step 2 角色动力：
- 角色卡片列表
- 每张卡片：名称、驱动力三角（表层/深层/灵魂）、弧线设计、关系网
- 可展开/折叠
- 操作：编辑单个角色、添加角色、删除角色

Step 3 角色初始状态：
- 角色列表，每个角色展开显示：物品、能力、身体/心理状态、位置、关系
- 表格或结构化表单形式

Step 4 世界观：
- 富文本显示/编辑区
- 分段显示：物理维度 / 社会维度 / 隐喻维度

Step 5 三幕结构：
- 三幕分栏或 Tab 切换
- 每幕显示：章节范围、摘要、关键转折点、反转、伏笔计划、悬念钩子
- 可编辑

**全局操作：**
- "从第 N 步开始生成"按钮
- 生成进度指示器（SSE 驱动）

### 章节蓝图页

**布局：** 表格视图 + 详情侧边栏

**表格列：**
- 章节号
- 标题
- 核心功能（advance/twist/reveal/setup/climax，彩色标签）
- 悬念密度（1-5，进度条或星级）
- 认知颠覆度（1-5，星级）
- 伏笔操作数量
- 状态（未生成/已生成/已编辑）

**点击某行 → 右侧展开详情面板：**
- 完整蓝图信息
- 伏笔操作列表（可编辑）
- 编辑/保存按钮

**顶部操作：**
- "生成蓝图"按钮（可指定起始章节）
- "重新生成选中批次"按钮
- 生成进度指示器

**悬念密度可视化：**
- 表格上方显示一条折线图，X 轴为章节号，Y 轴为悬念密度
- 直观展示节奏曲线

### 写作工作台

**布局：** 三栏

**左栏（章节导航）：**
- 章节列表（可折叠）
- 每章显示：章节号、标题、状态（draft/final）、字数
- 当前章节高亮
- 点击切换章节

**中栏（编辑区）：**
- 章节标题
- 富文本编辑器（Markdown 或所见即所得）
- 底部状态栏：字数统计、状态、最后保存时间
- 操作按钮：
  - "AI 生成本章"（触发流水线，流式显示生成过程）
  - "保存"（手动保存，触发章后管线）
  - "重新生成"（清除当前内容，重新生成）

**右栏（上下文面板，可折叠）：**
- Tab 切换：
  - 蓝图：当前章节的蓝图信息
  - 角色：出场角色的当前状态
  - 伏笔：活跃伏笔列表
  - 摘要：前几章的摘要
  - 流水线：当前章节的流水线中间产物

**AI 生成交互流程：**
1. 用户点击"AI 生成本章"
2. 按钮变为进度指示器，显示当前阶段（Plan → Compose → Draft → Commit）
3. Draft 阶段：文本在编辑器中逐字流式显示
4. Commit 阶段：右栏自动刷新角色状态和伏笔
5. 完成后编辑器切换为可编辑模式

### 角色管理页

**布局：** 卡片网格 + 详情面板

**每张角色卡片：**
- 角色名称
- 驱动力三角摘要
- 当前状态摘要（位置、身体/心理状态）
- 最后更新章节

**点击展开详情：**
- 完整角色信息（驱动力、弧线、关系网）
- 当前状态详情（物品、能力、状态、位置、关系）
- 状态历史时间线（按章节展示状态变化）

### 伏笔台账页

**布局：** 分组列表

**分组：**
- 活跃伏笔（open）
- 已解决伏笔（resolved）

**每条伏笔显示：**
- 描述
- 状态标签（彩色）
- 植入章节
- 最后推进章节
- 解决章节（如已解决）
- 相关角色

**操作：**
- 手动创建伏笔
- 编辑伏笔
- 手动标记为已解决

### LLM 模型配置页

**布局：** 表格 + 表单

**表格显示所有配置：**
- 名称
- 提供商
- 模型
- 任务类型
- 是否默认
- 操作：编辑 / 删除 / 测试连接

**新增/编辑表单：**
- 名称
- 提供商（下拉：OpenAI 兼容 / Anthropic）
- Base URL（OpenAI 兼容时显示）
- API Key（密码输入框）
- 模型 ID
- 任务类型（下拉：plan/draft/reflect/summarize/outline/blueprint/general）
- 参数：temperature（滑块）、max_tokens（数字）
- 设为默认（开关）
- "测试连接"按钮

## 通用组件

### SSE 流式显示组件

用于大纲生成、蓝图生成、章节生成等所有流式操作：
- 连接 SSE 端点
- 解析事件类型，更新对应 UI
- 显示进度指示器
- 支持取消操作
- 断线重连

### 状态标签组件

统一的彩色标签：
- draft → 灰色
- final → 绿色
- open → 蓝色
- resolved → 绿色
- advance → 橙色
- twist → 红色
- reveal → 紫色

### 确认对话框

用于删除、重新生成等破坏性操作的二次确认。
