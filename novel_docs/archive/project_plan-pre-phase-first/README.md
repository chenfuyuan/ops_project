# Legacy Project Plan Archive

该目录保存 `novel_docs/project_plan/` 在 phase-first 重组之前的历史规划资产。

## 归档说明

- 这些文档保留原有结构与文件名，用于追溯当时的思路和背景。
- 它们不再作为当前规划入口，也不应继续被视为后续 OpenSpec change 的直接起点。
- 当前有效的前期规划入口已经迁移到 [`novel_docs/project_plan/`](../../project_plan/00-overview.md)。

## 归档范围

- 旧总览、架构、数据模型、核心服务与阶段路线文档
- 旧 phase 目录中的 overview 与专题设计文档
- 旧的横切专题文档，例如 `common-components.md`

## 使用方式

- 如果需要理解早期方案背景，可按原路径查阅归档内容。
- 如果要继续推进新需求，请返回新的 roadmap 入口，再从对应 phase 的 requirement card 启动后续 change。
